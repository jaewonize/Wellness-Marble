# CLAUDE.md — Wellness Marble

## 상위 컨텍스트

### 서비스 개요
저속노화(Slow Aging) 지표를 관리하는 헬스케어 앱.
4개 카테고리(잘 움직이기/잘 쉬기/잘 가꾸기/잘 다스리기) 점수를
시각적으로 보여주는 그래픽 대시보드.

### 데이터 수집 방식
- 엔드 프로덕트: 시스템이 자동 수집/분석 (웨어러블, 셀피 분석)
- 유저가 직접 입력하지 않음
- PoC: 수동 입력으로 점수 변화에 따른 그래픽 반응을 시연

### PoC A 목적
점수 변화에 따라 그래픽이 반응하는 걸 보여주기 위한 시뮬레이터.
엔드유저용 앱 화면이 아닌, 디자이너/기획자가 실시간으로 화면 느낌을 확인하는 용도.

### PoC A 레이아웃 원칙
- **좌측 (모바일 화면)**: 엔드유저가 실제로 보게 될 화면. 컨트롤 UI 없음.
- **우측 여백 (컨트롤 패널)**: 시연자/디자이너용. 모바일 화면 밖 브라우저 여백에 배치.
  - 4개 카테고리 슬라이더
  - 트리거 버튼들 (점수 급락, 할로윈 특별 보상, 100회 방문 달성 등)
- 슬라이더/버튼 조작 → 좌측 모바일 화면 그래픽이 동적으로 반응

```
[브라우저 창]
┌─────────────────┬──────────────────┐
│                 │  🎛 Control Panel │
│  📱 모바일 화면  │  ── 잘 움직이기  │
│  (유저가 보는   │  ── 잘 쉬기      │
│   실제 화면)    │  ── 잘 가꾸기    │
│                 │  ── 잘 다스리기  │
│                 │                  │
│                 │  [트리거 버튼들]  │
└─────────────────┴──────────────────┘
```

### 전체 테마 목록 (PoC A)
- **Slow Terrarium** (지구 꾸미기)
- **Slow Museum** (갤러리)
- **Wellness Marble** (보드게임판) ← 이 폴더

## 프로젝트 개요
모노폴리/부루마불 스타일 1열 사각형 보드게임판 테마.
말이 보드 위를 유유자적 순환하는 속도가 저속노화 속도를 표현함.
후속 액션 유도 없는 순수 대시보드 성격.

## 기술 스택
- Canvas 2D API
- 순수 HTML/CSS/JS
- 단일 HTML 파일

## 파일 구조
```
wellness-marble/
├── CLAUDE.md
├── index.html                 # 메인 파일
├── lib/                        # GLTFLoader, meshopt decoder (로컬)
└── asset/bike/                 # 자전거 glb (원본 + .opt.glb 압축본)
```

## 핵심 원리
- 말 순환 속도 = 저속노화 속도
- 점수 높을수록 → 말이 천천히 여유롭게 순환
- 점수 낮을수록 → 말이 빠르게 순환
- 슬로우 테라리움 자전속도와 동일한 원리

## 보드 구조
```
[모서리1: Start]      — 주간 정산 지점
[Side1: 활동의 광장]  — 잘 움직이기 — 타일 7개
[모서리2: 황금열쇠]   — 긍정 이벤트 트리거
[Side2: 안식의 섬]    — 잘 쉬기     — 타일 7개
[모서리3: 시계탑]     — 노화시계 속도 표시
[Side3: 미학의 거리]  — 잘 가꾸기   — 타일 7개
[모서리4: 무인도]     — 앱 방문 오래됐을 때
[Side4: 사색의 언덕]  — 잘 다스리기 — 타일 7개
```

## 타일 이름 (구역별 7개)

| 구역 | 타일 이름 |
|------|---------|
| 활동의 광장 | 활력, 도약, 생동, 열정, 질주, 약진, 비상 |
| 안식의 섬 | 고요, 여백, 안온, 노을, 숨결, 평온, 여유 |
| 미학의 거리 | 우아, 색채, 감각, 섬세, 멋, 결, 빛 |
| 사색의 언덕 | 침묵, 통찰, 내면, 균형, 사유, 명상, 깨달음 |

## 슬롯 경제
- Start 칸부터 시계방향으로 슬롯 활성화
- 점수 상승 → 해당 구역 타일에 보상 아이템 드랍
- 점수 하락 → 가장 끝 타일부터 역순 잠금 + 아이템 자동 회수(보관함 이동)
- 타일당 1개 아이템만 전시, 나머지는 보관함

## 보상 로직
- 구역별 보상 풀 따로 관리
- 드랍된 아이템은 해당 구역 어느 타일에나 배치 가능 (타일-아이템 짝 없음)
- PoC에서는 랜덤 드랍
- 희귀도 등급 미정

## PoC 컨트롤 패널
모바일 화면 바깥에 별도 패널로 배치.

### 점수 조작
- 카테고리별 슬라이더 4개 (0~100)

### 트리거 버튼 9개

**긍정 트리거:**
1. 4개 카테고리 모두 상승 → 황금열쇠 칸 이동 + 보너스 이벤트
2. 개인 최고 기록 경신 → 해당 구역 특별 아이템 등장
3. 연속 상승 streak → 구역 전체 빛 효과

**중립 트리거:**
4. 주간 정산 → Start 칸 이동 + 주간 요약 카드
5. 구역 완성 → 해당 구역 완성 이펙트

**경고 트리거:**
6. 셀피 갱신 필요 → 잘 가꾸기 구역 안개 효과
7. 데이터 공백 → 해당 구역 타일 흐려짐
8. 점수 급락 → 슬롯 잠금 애니메이션
9. 앱 미방문 → 무인도 칸 이동
